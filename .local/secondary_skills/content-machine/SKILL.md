# Content Machine

Use this skill when:
- The user needs social posts, newsletters, or a content calendar.
- A brand voice needs to be matched or defined.
- The output must fit a specific platform rather than generic copy.

Workflow:
1. Confirm audience, platform, campaign goal, and tone.
2. Sample existing brand content when possible before drafting a new voice.
3. Write for platform mechanics such as hook length, scannability, CTA style, and posting format.
4. Use current facts, launches, or claims only when verified.
5. Deliver clean drafts that are ready to post, review, or repurpose.
