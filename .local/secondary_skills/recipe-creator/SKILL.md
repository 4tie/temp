# Recipe Creator

Use this skill when:
- The user wants a new recipe or meal idea.
- Available ingredients or dietary constraints define the solution.
- The output should be ready to cook or share.

Workflow:
1. Confirm servings, dietary rules, time budget, and available equipment.
2. Build a recipe that uses ingredients efficiently and avoids unnecessary complexity.
3. Include prep steps, cook steps, timing, and optional substitutions.
4. Add nutrition or storage notes only when they materially help.
5. Prefer plain markdown unless the user asks for a different format.
