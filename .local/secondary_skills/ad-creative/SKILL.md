# Ad Creative

Use this skill when:
- The user needs static paid-social or display ad concepts.
- A campaign needs new hooks, headlines, or creative directions.
- The deliverable should be reusable by design or marketing teams.

Workflow:
1. Confirm audience, offer, channel, format, and success metric.
2. Develop a small set of distinct concepts with different hooks.
3. Pair each concept with headline copy, support copy, CTA, and visual direction.
4. If the host supports image generation or design delegates, use them after the concept brief is clear.
5. Deliver concepts in plain markdown or repo-local assets, not editor-only surfaces.
