# AI Recruiter

Use this skill when:
- The user is hiring for a role and needs sourcing or screening help.
- A job description, scorecard, or interview plan needs work.
- Candidate communication needs drafting or review.

Workflow:
1. Confirm role, seniority, location, must-have skills, and timeline.
2. Build a scorecard before sourcing or screening.
3. Produce search guidance, screening notes, interview plans, and outreach drafts as separate artifacts.
4. If the host can access email or calendar tools, require explicit user confirmation before sending messages or scheduling anything.
5. Keep hiring recommendations evidence-based and note any missing context.
