# AI Secretary

Use this skill when:
- The user needs help with scheduling, agendas, follow-ups, or daily planning.
- A message or calendar change needs to be drafted clearly.
- Travel or task coordination needs a compact operational plan.

Workflow:
1. Confirm the objective, stakeholders, timing constraints, and desired output.
2. Draft messages and plans in plain language with the action request up front.
3. Keep calendars read-first and require explicit confirmation before creating, changing, or canceling an event.
4. If the host exposes connected communication tools, use them only after the user approves the exact action.
5. Return a clean next-step list so the user can execute or approve with minimal friction.
